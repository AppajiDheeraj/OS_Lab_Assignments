## About This Repository

This repository contains the complete set of **Operating Systems Laboratory programs** for the course **CS257** (Academic Year 2024).
It is designed to help students understand fundamental OS concepts through practical implementation using the **C programming language**.

The repository is structured in a clean and organized manner so that anyone can:

* Easily navigate through lab folders
* Compile and run programs using GCC
* Refer to questions and solutions together
* Revise concepts for exams and viva

## How to Compile and Run

### Step 1: Clone the repository

```bash
git clone https://github.com/your-username/OS_Lab_Assignments.git
```

### Step 2: Navigate to a lab folder

```bash
cd OS_Lab_Assignments/Lab_02
```

### Step 3: Compile the program

```bash
gcc filename.c
```

### Step 4: Run the executable

```bash
./a.out
```

*For Windows (MinGW):*

```bash
a.exe
```
